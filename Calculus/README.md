## 📖 О проекте

Интерактивный веб-тренажёр для проверки знаний по математическому анализу.  
Разработан для студентов МГУ (филиал в Душанбе) как инструмент самоподготовки к экзаменам.  
Оформлен в стиле **Deep Ocean** — тёмная тема с синими акцентами.

| ∑ Ряды | Числовые ряды · Сравнение · Даламбер/Коши · Знакочередующиеся · Степенные · Тейлор · Применения |

## ✍️ Открытый режим ответов

- Переключение **Закрытый / Открытый** прямо на странице раздела
- Только числовые и типизируемые ответы (кириллица отфильтрована автоматически)
- Автосохранение введённого ответа при переходе к следующему вопросу
- **Справочник форматов:** `3`, `-2`, `1/2`, `0.5`, `sqrt(2)`, `pi`, `e`, `ln(2)`, `e^2`, `-pi/2`, `1/sqrt(2)` и др.
- Нормализация: `1/2` = `0.5`, `sqrt4` = `2`, `pi` = `3.14159…`

## ⚔️ Система дуэлей

- Хост создаёт комнату → получает 6-значный код
- Гость вводит код → оба видят таймер обратного отсчёта
- Одинаковые вопросы для обоих (детерминированный генератор на основе кода)
- Раздел и уровень — на выбор; **смешанный режим** гарантирует вопросы из каждого раздела
- По окончании — сравнение результатов, реванш
- Защита: один аккаунт — одно устройство одновременно

## 🗂 Структура разделов

### ∫ Интегралы
- **Лёгкий** — табличные интегралы, линейные подстановки
- **Средний** — интегрирование по частям, тригонометрические подстановки
- **Сложный** — рациональные дроби, нестандартные приёмы

### f′(x) Производные
- **Лёгкий** — таблица производных, правило суммы
- **Средний** — произведение, частное, сложная функция
- **Сложный** — логарифмическое дифференцирование, многоуровневые цепочки

### ∑ Ряды и последовательности
- **Лёгкий** — понятие предела, геометрический ряд, необходимое условие
- **Средний** — признаки Даламбера, Коши, Лейбница, степенные ряды
- **Сложный** — равномерная сходимость, ряды Фурье, ряды Лорана

### lim Пределы
- **Лёгкий** — стандартные пределы, замечательные пределы
- **Средний** — правило Лопиталя, эквивалентные бесконечно малые
- **Сложный** — пределы функций нескольких переменных, нестандартные случаи

### dy/dx Дифференциальные уравнения
- **Лёгкий** — задачи Коши, уравнения с разделяющимися переменными
- **Средний** — линейные ОДУ первого порядка, уравнение Бернулли
- **Сложный** — ОДУ второго порядка, системы уравнений

## 📊 Как работает тест

1. Зарегистрируйся или войди по email
2. Выбери раздел → выбери режим (закрытый/открытый) и уровень сложности
3. Отвечай за отведённое время (автоматически рассчитывается по сложности)
4. После теста — детальный разбор ошибок с правильными ответами
5. Результаты сохраняются в облаке, XP начисляется автоматически
